## Microsoft COCO

用于目标检测识别分割等任务。
含有91种常见类型的物体（4岁孩子能轻易认出）
总计328k图像，2.5m个目标实例。
组建方式：开发了相应的类别检测，标记，分割的工具，由大量人群参与组建。
与PASCAL,ImageNet,SUN进行了详细的数据对比。
提供了基于DPM方法的baseline.
